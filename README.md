# Automation Bot Starter

## Overview
A clean starter template for building Telegram bots with a focus on maintainable structure, event-driven handling, and real-world automation patterns.

## What this shows
- Basic bot scaffolding and command handling
- Event-driven patterns for processing messages/actions
- Configuration and environment setup
- Patterns that scale to multiple workflows and integrations

## Tech
- Node.js / TypeScript
- Telegraf
- Optional: Redis (sessions/cache)
